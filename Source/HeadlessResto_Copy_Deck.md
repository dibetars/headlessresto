## 11. Copy Style Guide

### Voice

Confident and operator-first. We write the way a good head chef talks — direct, no fluff, outcome-led. Warm but not soft. Bold but not arrogant.

### Words to Never Use

| Avoid | Use instead |
|-------|-------------|
| Seamless | Smooth, fast, clean |
| Streamline | Cut down, simplify, reduce |
| Revolutionize | Change, fix, improve |
| All-in-one | One platform, unified, everything in one place |
| Game-changer | Just say what it actually changes |
| Empower | Give, let, help (or cut it entirely) |
| Leverage | Use |
| Solution | Platform, tool, software, system |

### Words We Love

| Use these | Why |
|-----------|-----|
| Offline | Our key differentiator — use it proudly |
| Sync / syncs | Technical credibility, implies reliability |
| Dispatch | Precise, operator-language for delivery |
| Queue | Real restaurant and tech vocabulary |
| Tight / tighter | Operator-speak for a well-run service |
| Rush | Evokes the lived restaurant experience |
| Bump | KDS terminology — signals deep product knowledge |
| No [X]. Ever. | Emphatic, absolute, memorable |

### Formatting Rules

- Em dash (—) over hyphen for asides and pauses. Never a double hyphen.
- Sentence case on all headings and buttons. Never title case or all caps in body.
- Oxford comma always.
- Contractions are fine (can't, you're, we'll) — they keep the tone warm.
- Max headline length: 8 words. Max subhead: 30 words. Max feature card: 35 words.
- Numbers under 10: write out (two taps, five locations). 10 and above: numerals (47 restaurants, $79/mo).
